O mapa prático: hospedagem grátis, código pronto e rotina de escrita

10 de julho de 2026

Para criar um site exatamente no estilo de Paul Graham ou John Gruber, você precisa seguir três passos fundamentais: escolher onde colocar o site de graça, estruturar o código de forma leve e definir a sua rotina de escrita.

Aqui está o mapa prático para colocar o seu projeto no ar hoje mesmo.

Passo 1: escolha a hospedagem gratuita e veloz

Como o seu site será feito apenas de texto e arquivos leves (HTML/CSS), você não precisa pagar por hospedagens pesadas como as de WordPress. Use plataformas focadas em desenvolvedores, que entregam uma velocidade imbatível de graça:

  • GitHub Pages: a melhor opção do mundo. Você guarda os seus arquivos de texto lá e o GitHub os transforma em um site acessível para qualquer pessoa.
  • Vercel ou Netlify: plataformas modernas onde você arrasta a pasta do seu site e ela entra no ar instantaneamente.

Passo 2: o código estrutural (pronto para usar)

Crie um arquivo chamado index.html e cole o código abaixo. Ele já vem configurado com uma tipografia bonita e limpa (estilo jornalístico), mas sem nenhuma firula visual que atrase o carregamento:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width, initial-scale=1.0">
    <title>Seu Nome - Ideias e Ensaios</title>
    <style>
        body {
            font-family: Georgia, serif; /* fonte clássica de leitura */
            max-width: 650px;    /* largura que não cansa os olhos */
            margin: 40px auto;   /* centraliza o site na tela */
            padding: 0 20px;
            line-height: 1.8;    /* espaçamento confortável */
            color: #111;         /* texto quase preto */
            background-color: #fcfcfc; /* fundo off-white */
        }
        h1 { font-size: 2.2em; margin-bottom: 5px; }
        h2 { font-size: 1.5em; margin-top: 40px; }
        .sub { color: #666; font-style: italic;
               margin-bottom: 40px; }
        ul { list-style-type: none; padding: 0; }
        li { margin-bottom: 15px; }
        a { color: #0066cc; text-decoration: none; }
        a:hover { text-decoration: underline; }
        hr { border: 0; border-top: 1px solid #eee;
             margin: 40px 0; }
    </style>
</head>
<body>

    <h1>Seu Nome</h1>
    <div class="sub">Pensamentos sobre negócios,
        tecnologia e cotidiano.</div>

    <p>Este é o meu canto na internet. Aqui eu escrevo
    de forma direta sobre o que estou aprendendo, criando
    e observando no mundo, sem distração de anúncios
    ou pop-ups.</p>

    <h2>Ensaios Recentes</h2>
    <ul>
        <li><strong>Jul 2026</strong> —
            <a href="artigos/primeiro-texto.html">O título do
            seu ensaio mais profundo aqui</a></li>
        <li><strong>Jun 2026</strong> —
            <a href="artigos/segundo-texto.html">Uma análise
            sobre o mercado ou tecnologia</a></li>
    </ul>

    <h2>Notas e Curadorias</h2>
    <ul>
        <li><strong>08/07</strong> —
            <a href="https://link-externo.com">Li esse artigo
            incrível hoje</a> e acho que a inteligência
            artificial vai mudar este setor por este motivo
            específico...</li>
    </ul>

    <hr>
    <footer>
        <p>Inscreva-se na minha <a href="#">Newsletter</a> |
        Contato: <a href="mailto:voce@email.com">
        voce@email.com</a></p>
    </footer>

</body>
</html>

Passo 3: a estratégia de conteúdo (como eles escrevem)

Para atrair o fluxo, seu site precisa seguir a mentalidade de um dos dois:

  1. Se você quer o estilo Paul Graham (ensaios): escreva textos longos (mais de 1.000 palavras) apenas quando tiver uma visão muito clara sobre um assunto. Publique poucas vezes (1 ou 2 vezes por mês), mas garanta que o texto seja tão bom que as pessoas queiram salvar nos favoritos.
  2. Se você quer o estilo John Gruber (notas diárias): acesse a internet de manhã, leia as notícias do seu nicho, escolha uma, coloque o link dela e comente com três frases sinceras dizendo por que aquela notícia importa (ou por que ela está errada). Faça isso todo santo dia.
